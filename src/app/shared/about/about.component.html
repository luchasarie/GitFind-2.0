SOBRE GIT_FIND
GIT_FIND é um projeto de Angular feito inicialmente pra um desafio de programação da Petize. Essa versão é um Re-Design da primeira versão, com um foco em um tema escuro, cores gradientes, e modernização dos layouts, além de um adicional de design responsivo para celulares.

Você pode encontrar o código disponível publicamente abaixo, tanto da versão 1.0 quanto da versão 2.0.
1.0 Repositório Github
Repositório do Git_Find 1.0 com instruções para instalação e build do projeto
1.0 WebApp Online
Versão online do Git_Find 1.0 hosteada no Firebase pra facíl acesso do aplicativo.
2.0 Wireframe Figma
Wireframing e Design Inicial da versão 2.0 no Figma.
2.0 Repositório Github
Repositório do Git_Find 2.0 com instruções para instalação e build do projeto
2.0 WebApp Online
Versão online do Git_Find 2.0 hosteada no Firebase pra facíl acesso do aplicativo.
